ZS TableSortable

Adds sortable behavior to ZSTable.

Constructor

new ZS TableSortable()

Source:
  • sort.js, line 4
Example
Add sortable attribute to every element suppose to have sorting listener then simply listen to "sort" event of entire table. 
				Additionally you can provide sorting-event (click by default) and sorting-elm ("a" by default) as attributes of your sorting elm.
