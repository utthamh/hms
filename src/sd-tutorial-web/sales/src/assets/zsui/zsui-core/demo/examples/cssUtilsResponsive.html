ZSUI core modules include several responsive utilies that can we used on different screen size as per the requirement.

We have many CSS utilities in core module but if we want to use these utilities on a specific screen size, we can put -lw, -mw, -sw and -xsw as a suffix to the required utility class.

-lw : large width - (min-width: 1200px)

-mw : medium width - (min-width: 992px) and (max-width: 1199px)

-sw : small width - (min-width: 768px) and (max-width: 991px)

-xsw : extra-small width - (max-width: 768px)

To check the impact of different utilities resize the window.
